Welcome to 'The Predictor' !

To predict an image you should follow the instructions:

1. open Postman
2. send a http request
  • type: POST
  • url: http://193.106.55.179:8888/isReceipt
  • body: add new row
    • key: convert the input from 'Text' to 'File' and write the word 'photo'
    • value: upload your image which you want to predict
3. Click Send
